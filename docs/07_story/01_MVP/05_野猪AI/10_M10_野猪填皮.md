## 技术设计要点

### 节点改造（boar.tscn）

```
Boar
- BodySprite (ColorRect) ❌ 移除
+ BodySprite (AnimatedSprite2D)
    - SpriteFrames（idle / walk）
```

### SpriteFrames 配置（美术清单附录 A）

| 动画名 | 源纹理 | hframes | fps | loop |
|---|---|---|---|---|
| `idle` | Boar idle | 6 | 8 | true |
| `walk` | Boar walk | 9 | 10 | true |

### PatrolState 动画选择

```gdscript
func enter(_msg := {}) -> void:
    host.body_sprite.play("walk")
func physics_process(delta):
    # 移动逻辑...
    host.body_sprite.flip_h = host.facing_dir < 0.0
```

### 可视化搭建协作（§12.4）

AnimatedSprite2D 动画配置触发 §12.4：AI 建 idle/walk 动画名 + 默认参数，用户拖入 Boar 帧纹理、精调 offset 脚底对齐。

## 验证步骤（§12.5）

1. **运行方式**：F5 启动 level.tscn。
2. **M10-1**：确认野猪精灵显示、脚底贴地。
3. **M10-2/3**：观察走动播 walk、转向/静止播 idle。
4. **M10-4**：左右走确认 flip_h 正确。

**预期**：野猪精灵走动有动画、方向正确。

**异常判定**：跳位 → offset 脚底对齐；动画不播 → SpriteFrames 帧配置。
