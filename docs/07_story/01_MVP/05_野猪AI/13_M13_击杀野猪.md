### 伤害链（玩家 Hitbox → 野猪 Hurtbox → 野猪 Health）

复用 M8 组件链路（反向）：
- 玩家 Hitbox（layer=玩家攻击区(4), mask=受击区(16)）命中帧激活
- 命中野猪 Hurtbox → 野猪 Health.take_damage(15)
- health_changed → 若 > 0：transition_to("hurt")；若 == 0：died → transition_to("dead")

### HurtState（野猪，架构 §4.4）

```gdscript
func enter(_msg := {}) -> void:
    host.body_sprite.play("hit")
    host.velocity = host.knockback_dir * host.stats.knockback
    _stun_timer = host.stats.hurt_stun
func physics_process(delta):
    host.velocity = host.velocity.move_toward(Vector2.ZERO, host.stats.friction * delta)
    _stun_timer -= delta
    if _stun_timer <= 0:
        machine.transition_to("chase")   # 还有血，回追击
```

### DeadState（架构 §4.4，Hit-Vanish 淡出）

```gdscript
func enter(_msg := {}) -> void:
    host.body_sprite.play("hit")
    host.hitbox.monitoring = false
    host.hurtbox.monitoring = false
    var tween := create_tween()
    tween.tween_property(host.body_sprite, "modulate:a", 0.0, host.stats.despawn_time)
    tween.tween_callback(host.queue_free)
```

### 数值校验

- 野猪 max_health = 30，玩家 attack_damage = 15 → 2 击致死（30 - 15 - 15 = 0）。

### 关键决策

- **复用 M8 伤害链**（ADR-10）：Hitbox/Hurtbox/Health 组件跨实体复用，验证场景化设计价值。
- **Hit-Vanish 淡出 + queue_free**（架构 §4.4）：`is_instance_valid` 检查后再 queue_free（宪法 §4.2）。
- **移除 M8 静态靶**：真实野猪替代，避免冗余。

### 可视化搭建协作（§12.4）

## 验证步骤（§12.5）

1. **运行方式**：F5 启动 level.tscn（确认 M8 静态靶已删）。
2. **AC7**：对野猪挥剑 2 次，确认第 2 次后野猪死亡。
3. **M13-1/2**：观察 hit 动画 + 淡出 + 消失。
4. **M13-3**：第 1 击后观察野猪短暂硬直 + 击退。
5. **M13-4**：确认场景无静态靶残留。

**预期**：2 击致死，野猪淡出消失，无静态靶。

**异常判定**：1 击死/3 击不死 → max_health/attack_damage 数值；不淡出 → DeadState tween；撞空气 → 野猪 Hurtbox layer（16）与玩家 Hitbox mask（16）。
