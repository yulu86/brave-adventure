## 用户故事

**As a** 玩家，
**I want** 游戏中按 Esc（或手柄 Start）呼出暂停菜单，可选继续/重玩/返回标题，
**so that** 能随时暂停处理事务、重试或退出，且暂停时菜单仍可交互、BGM 降至 30%。

## 技术设计要点

### 新增场景与脚本（架构 §3.3 / ADR-8）

```
scenes/pause_menu.tscn   class_name PauseMenu (CanvasLayer, layer=100, process_mode=WHEN_PAUSED, 初始 visible=false)
├── DimOverlay (ColorRect, 全屏半透明黑)
├── Panel (PanelContainer, 居中)
│   └── VBoxContainer
│       ├── TitleLabel ("已暂停")
│       └── ButtonGroup
│           ├── ResumeButton ("继续")    # grab_focus 入口
│           ├── RetryButton ("重玩本关")
│           └── QuitButton ("返回标题")
scripts/ui/pause_menu.gd
```

### pause_menu.gd 契约（架构 §3.3）

```gdscript
class_name PauseMenu
extends CanvasLayer

signal resume_requested
signal retry_requested
signal quit_requested

func open() -> void:
    visible = true
    $Panel/.../ResumeButton.grab_focus()
func close() -> void:
    visible = false
func _on_resume_pressed() -> void: resume_requested.emit()
func _on_retry_pressed() -> void: retry_requested.emit()
func _on_quit_pressed() -> void: quit_requested.emit()
```

### GameManager 挂载与处理（架构 §3.1 / §8.1）

```
GameManager
+ PauseMenu (pause_menu.tscn 实例)   # 根自带 CanvasLayer(layer=100, WHEN_PAUSED)
```

```gdscript
# game_manager.gd
@onready var pause_menu: PauseMenu = $PauseMenu
func _unhandled_input(event: InputEvent) -> void:
    if event.is_action_pressed("pause"):
        if get_tree().paused:
            _resume()
        else:
            _pause()
func _pause() -> void:
    get_tree().paused = true
    pause_menu.open()
    set_bgm_volume_db(linear_to_db(0.3))   # 降至 30%
func _resume() -> void:
    get_tree().paused = false
    pause_menu.close()
    set_bgm_volume_db(0.0)
func _ready():
    pause_menu.resume_requested.connect(_resume)
    pause_menu.retry_requested.connect(_on_retry)
    pause_menu.quit_requested.connect(_on_quit)
func _on_retry() -> void:
    get_tree().paused = false
    get_tree().reload_current_scene()
func _on_quit() -> void:
    get_tree().paused = false
    get_tree().change_scene_to_file("res://scenes/main_menu.tscn")
```

### 输入

`project.godot [input]` 新增 `pause`（Esc + 手柄 Start）。GameManager `_unhandled_input` 捕获（跨场景）。

### 关键决策

- **PauseMenu 独立自包含子场景**（ADR-8）：根自带 CanvasLayer(layer=100) + process_mode=WHEN_PAUSED，暂停行为内聚。
- **process_mode=WHEN_PAUSED**：暂停时菜单仍可交互，其余节点 PAUSABLE 被暂停；GameManager process_mode=ALWAYS（M18）保证能处理输入。
- **按钮向上发信号**（宪法 §1.2/§5.1）：GameManager 连接处理，菜单不自执行场景切换。
- **BGM 降至 30%**（GDD §8.6）。

### 可视化搭建协作（§12.4）

Panel/按钮布局、DimOverlay 透明度属精调项。

## 验证步骤（§12.5）

1. **运行方式**：F5 进关卡后按 Esc。
2. **AC13/M21-1**：确认呼出暂停菜单，玩家/野猪停止，"继续"按钮聚焦。
3. **M21-2**：点"继续"，确认恢复 + BGM 恢复。
4. **M21-3**：点"重玩本关"，确认关卡重载、状态重置。
5. **M21-4**：点"返回标题"，确认切回主菜单。
6. **BGM**：暂停时确认 BGM 降至 30%。

**预期**：Esc 暂停，三按钮生效，BGM 正确切换。

**异常判定**：暂停后菜单不可点 → process_mode=WHEN_PAUSED 未设；暂停后游戏仍动 → 其余节点 process_mode 未 PAUSABLE；返回标题卡死 → quit 处理。
