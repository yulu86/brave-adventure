## 技术设计要点

### MineEntrance（架构 §3.4 / §2.1）

```
scenes 内或 level.tscn:
MineEntrance (Area2D, layer=触发区(64), mask=玩家(1))  class_name MineEntrance
└── CollisionShape2D (RectShape2D 触发区)
└── 视觉: Buildings.png 拱门瓦片 + 黑色填充（§12.4 精调）
scripts/level/mine_entrance.gd
```

> 触发区 layer_7 = 64，玩家 layer_1 = 1。

### mine_entrance.gd 契约

```gdscript
class_name MineEntrance
extends Area2D

signal level_completed

func _ready() -> void:
    body_entered.connect(_on_body_entered)
func _on_body_entered(body: Node2D) -> void:
    if body.is_in_group("player"):
        level_completed.emit()
```

### LevelController（架构 §3.4 / §7.1 / §8.3）

```
level.tscn:
LevelController (Node)  class_name LevelController
scripts/level/level_controller.gd
```

```gdscript
class_name LevelController
extends Node

@onready var mine_entrance: MineEntrance = %MineEntrance
@onready var player: Player = %Player

func _ready() -> void:
    mine_entrance.level_completed.connect(_on_level_completed)
    player.health.died.connect(_on_player_died)   # M23 用
func _on_level_completed() -> void:
    GameManager.on_level_completed()   # 转发 → 切 win_screen
```

### 胜利画面（GDD §9.1，shader 留 M25）

```
scenes/win_screen.tscn
└── WinScreen (Node2D)
    ├── Background (复用关卡背景)
    ├── ContentUI (CanvasLayer layer=100)
    │   ├── TitleLabel "🎉 突围成功！"
    │   ├── SubtitleLabel "你抵达了矿洞入口"
    │   └── ButtonGroup (再玩一次 / 返回标题)
    └── WinController (脚本)
scripts/ui/win_screen.gd
```

> M22 先做 UI 与流程，金色光波 shader 留 M25（本里程碑 GlowRect 可留空或占位）。

### GameManager 场景调度（架构 §8.1）

```gdscript
func on_level_completed() -> void:
    # BGM 淡出（GDD §8.6）
    change_scene_to_file("res://scenes/win_screen.tscn")
```

### 关键决策

- **MineEntrance 只发信号**（架构 §2.1）：不含胜负逻辑，LevelController 转发。
- **LevelController 判胜负**（架构 §8.3）：管本关 Player.died / MineEntrance，转发 GameManager。
- **触发区层隔离**（layer_7）：与伤害链隔离，过关不误触受击。
- **矿洞入口视觉**（GDD §3.4）：Buildings 拱门 + 黑色填充，§12.4 精调。

### 可视化搭建协作（§12.4 强制关卡）

矿洞入口拱门瓦片选型/黑色填充、win_screen 布局属精调项。

## 验证步骤（§12.5）

1. **运行方式**：F5 进关卡，走到右端矿洞入口。
2. **AC10**：进入矿洞入口，确认切到胜利画面。
3. **M22-1**：确认矿洞入口视觉（拱门+黑洞）。
4. **M22-2**：确认胜利画面标题/副标题/按钮。
5. **M22-3**：点"返回标题"/"再玩一次"，确认生效。

**预期**：走到终点→胜利画面，按钮生效。

**异常判定**：不触发 → MineEntrance layer/mask（触发区 64/玩家 1）、玩家 group；不切场景 → GameManager 调度。
